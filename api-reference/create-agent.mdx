### Instructions
1. Upload an image of the face you want to use to create avatar.
2. Wait for the avatar to be created. This can take a couple of hours. If you don't want to wait, use one of our [Default Faces](/api-reference/preset-faces)
3. Find your avatar in the [Your Avatars](https://app.simli.com) section.
