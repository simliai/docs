### Instructions
1. Upload an image of the face you want to use to create avatar.
2. Image width/height should be close to 1:1 (e.g., 512x512 pixels).
3. Click on the "Generate Preview" button and see that the preview video looks okay.
4. If you are happy with the preview, click on the "Create Avatar" button.
5. Wait for the avatar to be created, this may take a few minutes.
6. Find your avatar in the [Your Avatars](https://app.simli.com) section.
7. There you can customize your avatar's name, first message, system prompt, and voice.
