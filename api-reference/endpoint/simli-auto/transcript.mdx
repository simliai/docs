---
title: "/auto/transcript/{session_id}"
openapi: "GET /auto/transcript/{session_id}"
description: "Retrieve the conversation transcript for a specific session if transcript generation was enabled. If using the [auto/start/configurable](/api-reference/endpoint/simli-auto/start-configurable) API, `createTranscript` must be set to `true`. If using the [sessionToken](api-reference/endpoint/simli-auto/token)-based API, `createTranscript` must be set to `true` in the token configuration."
